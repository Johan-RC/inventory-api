## 📌 Descripción

Este proyecto corresponde a una aplicación completa de inventario desarrollada como actividad académica.

El sistema está dividido en **3 repositorios independientes**:

- Backend
- Frontend
- Base de datos

Además, cuenta con despliegue en **Render** y automatización mediante **GitHub Actions**.
