## 🧠 Descripción

✨ Sistema completo de gestión de inventario que permite administrar:

📦 Productos  
🏷️ Categorías  
🚚 Proveedores

## 🔗 API Endpoints

📂 Categorías  
GET /api/categories  
POST /api/categories  
PUT /api/categories/{id}  
DELETE /api/categories/{id}  

🚚 Proveedores  
GET /api/suppliers  
POST /api/suppliers  
PUT /api/suppliers/{id}  
DELETE /api/suppliers/{id}  

📦 Productos  
GET /api/products  
POST /api/products  
PUT /api/products/{id}  
DELETE /api/products/{id}
